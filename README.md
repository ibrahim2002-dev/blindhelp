# BlindHelp

A Vite + React project for a vision assistance app.

## Deployment

This project is configured for GitHub Pages deployment at:

`https://ibrahim2002-dev.github.io/blindhelp/`

### Prepare

1. Install dependencies:

```bash
npm install
```

2. Build the project:

```bash
npm run build
```

3. Deploy to GitHub Pages:

```bash
npm run deploy
```

### Notes

- The app uses the browser camera and may require HTTPS for camera access.
- GitHub Pages serves the production build from `dist/`.
- The `vite.config.ts` file is set up with the correct base path for this repository.
